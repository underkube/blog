Another week goes by, another collection of links I found interesting...

## Infrastructure and Operating Systems

* [**FreeBSD Officially Supported in OCI Runtime Specification v1.3**](https://freebsdfoundation.org/blog/freebsd-officially-supported-in-oci-runtime-specification-v1-3/) - The Open Container Initiative (OCI) Runtime Specification v1.3 officially includes FreeBSD support. This major milestone positions FreeBSD as a first-class platform for cloud-native workloads, leveraging its existing jails technology for container isolation.
* [**Incus: Next-Generation Container and VM Manager**](https://linuxcontainers.org/incus-os/) - Incus is a next-generation system container, application container, and virtual machine manager that provides a user experience similar to a public cloud. It allows users to easily mix and match both containers and VMs, sharing the same underlying storage and network.
* [**Giving a Rootful Podman Container its Own IP**](https://www.blog.montgomerie.net/posts/2025-10-18-giving-a-rootful-podman-container-its-own-ip/) - A technical post detailing methods and configurations for assigning a dedicated IP address to a rootful Podman container.
* [**Simple but IPv6 Capable Home Router with OpenBSD 7.7**](https://www.blog.montgomerie.net/posts/2025-10-11-setting-up-a-very-simple-but-ipv6-capable-home-router-with-openbsd-7.7/) - A guide on setting up a basic yet fully functional home router using OpenBSD 7.7.
* [**SUSE Provides U-Boot Support For The Raspberry Pi 5**](https://news.opensuse.org/2025/11/04/raspberrypi5-opensuse/) - SUSE's hardware enablement team has worked to provide proper U-Boot support for the Raspberry Pi 5 Single Board Computer.
* [**Chromium Sync with Google Account**](https://ohyaan.github.io/tips/enabling_chromium_to_sync_with_google_account/#4-restart-chromium) - A technical tip on enabling Google Account syncing functionality within the Chromium web browser.

## Software and Web Development

* [**Your URL Is Your State**](https://alfy.blog/2025/10/31/your-url-is-your-state.html) - A blog post discussing the often-overlooked power of URLs as first-class state containers in modern web applications, enabling shareability and bookmarkability without needing complex state management tools.
* [**You Don't Need Anubis**](https://lobste.rs/s/gig2wt/you_don_t_need_anubis) - A discussion about alternative approaches to DNS servers, referencing the project Anubis and its features like DNS Blocklist (DNSBL) checking for known bad actors.
* [**Linking Directly to Web Page Content**](https://alfy.blog/2024/10/19/linking-directly-to-web-page-content.html) - An article discussing methods and standards for creating deep links that target and highlight specific text fragments within a web page.
* [**Nextcloud Performance is Terribly Slow**](https://ounapuu.ee/posts/2025/11/03/nextcloud-slow/) - A troubleshooting post discussing common causes of slow Nextcloud performance, such as unoptimized caching, background tasks, and issues related to specific storage backends like GlusterFS.

## AI, Research, and Benchmarking

* [**ollama-benchmark**](https://github.com/larryhopecode/ollama-benchmark) - A handy tool to measure the performance and efficiency of Large Language Model (LLM) workloads running on Ollama, featuring various tests like chat speed, embedding, model loading speed, and quality judging.
* [**Exploring a Space-Based Scalable AI Infrastructure System Design (Project Suncatcher)**](https://research.google/blog/exploring-a-space-based-scalable-ai-infrastructure-system-design/) - A Google research paper detailing Project Suncatcher, a design for a scalable AI infrastructure utilizing a constellation of satellites equipped with TPUs and high-bandwidth free-space optical communication.
