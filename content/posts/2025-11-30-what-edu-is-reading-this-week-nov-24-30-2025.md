## AI, LLMs & Automation

* [**Model Context Protocol (MCP) Introduction**](https://modelcontextprotocol.io/docs/getting-started/intro): Documentation for the **Model Context Protocol**, an open standard designed to enable AI agents (LLMs) to interact securely and predictably with external tools, APIs, and data.

* [**Jules: The Autonomous Agent CLI Extension**](https://jules.google/) / [**Jules in 90 seconds**](https://www.youtube.com/watch?v=M2G27_B7BBM) / [**Jules GitHub Repo**](https://github.com/gemini-cli-extensions/jules): The official page (and a GitHub repository for a Gemini extension) of Jules, an **asynchronous coding sidekick**, letting developers delegate long-running tasks.

* [**openSUSE systemd-mcp**](https://github.com/openSUSE/systemd-mcp): The openSUSE project implementing an MCP server to allow AI assistants to interact with and manage Linux systems via `systemd` (e.g., start/stop systemd services).

* [**MCP Host CLI (mark3labs/mcphost)**](https://github.com/mark3labs/mcphost): A CLI host application that manages connections between LLMs (like Claude, Gemini, Ollama) and external tools through the Model Context Protocol.

* [**Turning Gemini CLI into a Multi-Agent System**](https://aipositive.substack.com/p/how-i-turned-gemini-cli-into-a-multi): A guide detailing how to transform the Gemini CLI into a multi-agent orchestration system using clever prompting techniques alone.

* [**Gemini CLI Tips and Tricks**](https://github.com/addyosmani/gemini-cli-tips): A helpful repository collecting best practices and custom command examples for using the Gemini command-line interface efficiently.

* [**CERN's General Principles for AI Use**](https://home.web.cern.ch/news/official-news/knowledge-sharing/general-principles-use-ai-cern): The official ethical and operational guidelines set by CERN for the use of artificial intelligence within the organization.

## Operating Systems & Linux Kernel

* [**FreeBSD 15.0 Release Notes**](https://www.freebsd.org/releases/15.0R/relnotes/): The official release notes for the new major version of FreeBSD.

* [**KDE Goes All-In on Wayland**](https://blogs.kde.org/2025/11/26/going-all-in-on-a-wayland-future/): KDE's official blog post reaffirming their commitment to making the Wayland protocol the primary and default display server for the Plasma desktop environment.

* [**ZFS-Enabled Disaster Recovery Virtualization**](https://klarasystems.com/articles/zfs-enabled-disaster-recovery-virtualization/): A guide on leveraging ZFS snapshots and replication for efficient disaster recovery and live migration of virtual machines.

* [**Building an IPv6-Focused OpenBSD Home Router**](https://blog.infected.systems/posts/2024-12-07-building-an-ipv6-focused-openbsd-home-router/?s=09): A technical guide on setting up a robust, security-focused home network gateway using OpenBSD with an emphasis on IPv6 configuration.

* [**Macbook 12' (2015, 2016, 2017) Omarchy Linux Installation Notes**](https://gist.github.com/wey-gu/0996ef17742dc5002eccea9391b62e9f): A Gist containing configuration and instructions to improve certain MacBook models running Omarchy.

* [**GNOME Shell Extension: Activate GNOME**](https://github.com/isjerryxiao/gnome-shell-extension-activate-gnome): A minimal GNOME Shell extension that allows users to mimic one of the best Windows 11 features, the activation watermark!.

* [**Aluminium OS: Android for PCs**](https://www.androidauthority.com/aluminium-os-android-for-pcs-3619092/): An article discussing Aluminium OS the alleged new Google's operating system to replace Chrome OS.

## Cloud Native & DevOps

* [**kubeswitch**](https://github.com/danielfoehrKn/kubeswitch): A powerful command-line tool for quickly switching between different Kubernetes cluster contexts using fuzzy finding.

* [**git-remote-s3**](https://github.com/awslabs/git-remote-s3): An AWS Labs project that enables using **Amazon S3 as a Git remote** and LFS server, providing a serverless option for hosting repositories.

* [**Canonical Chisel**](https://github.com/canonical/chisel) / [**Red Hat Hummingbird**](https://thenewstack.io/hummingbird-red-hats-answer-to-alpine-ubuntu-chiseled-wolfi/): Ultra-minimal container images from Canonical and Red Hat.

* [**Penpot**](https://github.com/penpot/penpot): The open-source design and prototyping tool for design and code collaboration (an alternative to Figma/Sketch).

* [**RESTerm**](https://github.com/unkn0wn-root/resterm): A terminal-based client for working with HTTP, GraphQL, gRPC, and WebSockets/SSE, built to be keyboard-first and offline-friendly.

* [**Disk Burn-In and Testing Script**](https://github.com/Spearfoot/disk-burnin-and-testing): A collection of scripts for stress-testing new hard drives and SSDs to catch early failures (burn-in testing).

## Programming & Learning

* [**One OS Thread, Millions of Goroutines**](https://shehroztalks.medium.com/one-os-thread-millions-of-goroutines-the-magic-of-gos-scheduler-explained-5144ec61f3a8): A detailed explanation of the Go runtime scheduler, detailing how it manages concurrency efficiently using the M:N scheduler model.

* [**Linux Kernel Explorer**](https://reverser.dev/linux-kernel-explorer): An interactive online tool that allows users to browse and study the Linux kernel source code with rich cross-referencing capabilities.

* [**Wails**](https://github.com/wailsapp/wails): A framework for building cross-platform desktop applications using Go (backend logic) and standard web technologies (frontend UI).

* [**Binary Exploitation Guide**](https://guyinatuxedo.github.io/): A comprehensive, structured guide intended for newcomers wanting to learn the fundamentals of binary exploitation and reverse engineering.

* [**We stopped roadmap work for a week and fixed 189 bugs**](https://lalitm.com/fixits-are-good-for-the-soul/): An article on the benefit of allocating time technical cleanups and fixes, improving both code quality and developer morale.

## Security & Privacy

* [**Exposed By Default**](https://neberej.github.io/exposedbydefault/): A scary demonstration showing exactly how much data (fingerprints, hardware info, network data) your web browser gives away automatically on every page load.

* [**If OpenSSL Were a GUI**](https://smallstep.com/blog/if-openssl-were-a-gui/): A blog post visualizing how the complex commands and processes of using OpenSSL for PKI (Public Key Infrastructure) might look in a modern graphical interface.

* [**Hacking the Agent (HackTheAgent.com)**](https://hacktheagent.com/): LLM Security Challenge.

* [**Google Antigravity Exfiltrates Data**](https://www.promptarmor.com/resources/google-antigravity-exfiltrates-data): Security research detailing a data exfiltration vulnerability found in Google's Antigravity AI development platform.

* [**Chameleon Ultra**](https://github.com/RfidResearchGroup/ChameleonUltra?tab=readme-ov-file): A powerful, open-source pocket-sized device for reading, emulating, and cloning various NFC and RFID tags.
