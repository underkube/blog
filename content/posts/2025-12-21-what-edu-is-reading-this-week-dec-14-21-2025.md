Here is the roundup of links for this week:

## Cloud & Infrastructure

* [**Kubernetes v1.35 Release**](https://kubernetes.io/blog/2025/12/17/kubernetes-v1-35-release/) - The official announcement for the latest version of Kubernetes.
* [**Docker Hardened Images**](https://www.docker.com/products/hardened-images/) / [**Catalog**](https://github.com/docker-hardened-images/catalog): Docker has made its catalog of hardened container images free to use, designed to secure the software supply chain.
* [**Why you can't ping a Kubernetes Service**](https://medium.com/@danielepolencic/learn-why-you-cant-ping-a-kubernetes-service-dec88b55e1a3) - A technical explanation of why standard ICMP pings don't work with K8s ClusterIP services.
* [**OVHCloud Kubernetes**](https://academy.fpblock.com/blog/ovhcloud-k8s/) - A guide or discussion regarding Kubernetes baremetal clusters on OVHCloud.
* [**Hive**](https://github.com/mulgadc/hive) - An open source, AWS-compatible platform (EC2, VPC, S3, and EBS-like services).

## DevOps & Tools

* [**GitHub Actions Pricing Saga**](https://github.blog/changelog/2025-12-16-coming-soon-simpler-pricing-and-a-better-experience-for-github-actions/) / [**Postponed**](https://pricetimeline.com/news/189): GitHub announced a new pricing model for Actions (including charges for self-hosted control planes), but subsequent reports indicate the price increase for self-hosted runners has been postponed.
* [**Scanopy**](https://github.com/scanopy/scanopy) - Previously known as NetVisor, this is a tool for network visualization.
* [**Charles Proxy**](https://www.charlesproxy.com/) - An HTTP proxy / HTTP monitor / Reverse Proxy that enables a developer to view all of the HTTP and SSL / HTTPS traffic between their machine and the Internet.
* [**Buildbot**](https://buildbot.net/) - An open-source continuous integration framework based on Python.
* [**Atomic**](https://github.com/flora131/atomic) / [**Atomic (video)**](https://www.youtube.com/watch?v=Lq8-qzGfoy4): Atomic: Automated Procedures and Memory for AI Coding Agents ([Davide Rutigliano](https://www.linkedin.com/in/davide-rutigliano/) shared this directly to me to post it here, thanks! :D)
* [**QJP**](https://github.com/plainas/qjp) - "Quick JSON Picker" - an interactive command-line menu for filtering and selecting JSON objects.

## Security

* [**First Linux Rust CVE**](https://www.phoronix.com/news/First-Linux-Rust-CVE) - A significant milestone: the first CVE assigned to Rust code within the Linux kernel, though it reportedly only causes a crash.
* [**North Korean Infiltrator at Amazon**](https://www.tomshardware.com/tech-industry/cyber-security/north-korean-infiltrator-caught-working-in-amazon-it-department-thanks-to-lag-110ms-keystroke-input-raises-red-flags-over-true-location) - An IT worker was caught due to a consistent 110ms keystroke lag, revealing their true location.
* [**My server started mining Monero**](https://blog.jakesaunders.dev/my-server-started-mining-monero-this-morning/) - A personal account of discovering a server compromise and unauthorized crypto mining.
* [**Jack.bio License Plate**](https://www.jack.bio/blog/licenseplate) - A post about acquiring a rare license plate using TypeScript.

* [**ZFS status on NetBSD**](https://mail-index.netbsd.org/netbsd-users/2025/12/19/msg033689.html) - A message from the NetBSD users mailing list explaining the current status of ZFS on NetBSD.
* [**DIY Ambilight Guide**](https://houndhillhomestead.com/diy-ambilight/) / [**LG webOS Setup**](https://github.com/satgit62/How-to-Install-and-set-up-Ambilight-on-LG-webOS?tab=readme-ov-file) / [**Kernel Drivers**](https://github.com/throwaway96/webos-kernel-drivers): A collection of resources for building a DIY ambient light system, specifically targeting LG webOS TVs.
